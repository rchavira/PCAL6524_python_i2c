# Installing

Stable library from PyPi:

* Just run `python3 -m pip install pcal6524`

Latest/development library from GitHub:

* `git clone https://github.com/rchavira/pcal6524-python`
* `cd pcal6524-python`
* `sudo ./install.sh --unstable`
